A Python-based desktop translator built with Tkinter. 

### Features:
- Supports **Arabic** (with proper RTL and character shaping).
- Supports **Japanese**, English, French, and more.
- Modern Flat UI design.
- One-click copy to clipboard.

### How to Run:
1. Install requirements: `pip install deep-translator arabic-reshaper python-bidi pyperclip`
2. Install Tkinter: `sudo apt install python3-tk`
3. Run: `python3 translator.py`
